# Pacis_jr-s3Mmcp23017

Standalone PACIS Jr hardware version using:

- Original Junior PACIS logic
- ESP32-WROOM button and RS485 wiring
- MCP23017 for all panel LED outputs
- Two ULN2803A drivers for 12 V illuminated push-button LEDs
